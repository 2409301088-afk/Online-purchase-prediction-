# 🛒 Online Purchase Intent Prediction

## 📌 Description
This project predicts whether a user will make an online purchase based on their browsing behavior using Machine Learning.

---

## 🎯 Objective
To classify user sessions into:
- Purchase
- No Purchase

---

## 🧠 Model Used
- Random Forest Classifier

---

## 📊 Features Used
- PageValues
- BounceRates
- ExitRates
- ProductRelated
- VisitorType
- Weekend

---

## 🌐 Web App
A simple web interface is built using Streamlit where users can input data and get predictions.

---

## ⚙️ How to Run

1. Install libraries:
pip install -r requirements.txt


2. Train model:


python model.py


3. Run app:


streamlit run app.py


---

## 📁 Files
- app.py → UI
- model.py → ML model
- dataset.csv → Data
- model.pkl → Saved model
